# Tic Tac Toe Game 🎮

A simple command-line Tic Tac Toe game developed in Python. This project allows two players to play the classic Tic Tac Toe game in the terminal with winner and draw detection.

## Features

* Two-player gameplay (X and O)
* Interactive command-line interface
* Winner detection for rows, columns, and diagonals
* Draw detection
* Input validation to prevent invalid moves
* Prevents players from selecting already occupied positions

## Technologies Used

* Python 3

## Gameplay

* The game starts with Player X.
* Players take turns entering a position from 1 to 9.
* The first player to align three marks horizontally, vertically, or diagonally wins.
* If all positions are filled and no player wins, the game ends in a draw.
